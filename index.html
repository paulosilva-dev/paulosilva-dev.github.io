BrickBreaker Evolution

I remade my previous game Brickbreaker in Unity 3D.
The base is similar but I added a few new features. And made the game prettier!
Unity really adds up a lot of flexibility =) I tried to use Unity’s physics engine for the ball but the bounces always looked a bit “off”, the collision detection was not perfect. So I ended up using a more hands on approach and handling the collisions directly. Made in C# and Unity 3D.

You can download it here: download BrickBreaker Evolution
music: Valiant by EnV

Planet Defender

I made game based on space invaders game but with a twist.
I was always curious about infinite games and random generation and wanted to try it out.
The waves are randomly generated based on some base patterns and the overall speed is increased over time.
Made with C# and XNA.

You can download and give it a go: download Planet Defender
You’ll need the XNA framework : xna
music: Space Invaders Remix by Caudex
